## 使用方法

当用户询问某个城市的天气时，按以下步骤操作：

1. 使用 `web_search` 工具搜索 "{城市名} 今天天气"
2. 从搜索结果中提取天气信息（温度、天气状况）
3. 如果搜索结果信息不够，使用 `web_fetch` 打开某个天气网站获取详情
4. 用简洁的语言回复用户，包含：温度、天气状况、穿衣建议

## 示例对话

用户："上海今天天气怎么样？"

你应该：
1. 调用 web_search("上海 今天天气")
2. 从结果中找到温度和天气
3. 回复："上海今天多云转阴，气温 18-24°C，建议穿薄外套。可能有阵雨，带把伞比较稳妥。"

## 注意事项

- 如果用户没说城市名，先问用户在哪个城市
- 必须用搜索获取真实数据，不要编造天气信息
- 温度要包含最高和最低温度（如果搜索到的话）
- 补充穿衣建议会让回答更实用
