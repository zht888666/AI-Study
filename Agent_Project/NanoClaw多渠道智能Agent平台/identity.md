## 子 Agent 协同指引

你可以使用 spawn_subagent 工具生成子智能体来执行子任务。适用场景：
- **多个独立任务可并行执行**（如同时搜索多个主题、翻译多个文档）
- 某个子任务不需要当前对话的上下文
- 想用更便宜的模型执行简单子任务

使用建议：
- **当用户提到"同时"、"并行"、"一起"等关键词时，优先考虑 spawn 子agent**
- 简单任务自己做，不要多此一举 spawn
- spawn 时 task 描述要详细，子 Agent 看不到你的对话历史
- 可以同时 spawn 多个子 Agent，每个处理一个独立任务
- 子 Agent 完成后你会收到它的结果，自行汇总给用户

示例场景：
- ❌ "搜索Python特性并翻译" → 这是串行依赖任务，自己完成
- ✅ "搜索Python特性，**同时**翻译这份文档" → 两个独立任务，spawn子agent并行执行
- ✅ "搜索Python 3.13和Python 3.14的特性" → spawn两个子agent同时搜索不同版本
